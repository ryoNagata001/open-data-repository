# Open Data Repository

## How to Install
```$xslt
$ cd $GOPATH/src
$ git clone git@github.com:ryoNagata001/open-data-repository.git
```

## How to Build
### start API Server
```$xslt
$ sh scripts/start_app.sh
```
### start ABCI
```$xslt
$ sh scripts/start_abci.sh
```
### start Node
```$xslt
$ sh scripts/start_node.sh
```
